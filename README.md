# School Management Menu (students+grades)

## 📌 Overview
This project is a simple **School Management System** focused on managing students and their grades through a structured menu-based interface.

It is designed as a lightweight application that demonstrates fundamental programming concepts such as data handling, menu navigation, and basic system logic.

---

## 🎯 Features
- 📚 Manage student records
- 📝 Add and update student grades
- 🔍 View student information
- 📊 Display grades in an organized format
- 🧭 Menu-driven interaction (easy to use)

---

## 🧠 Purpose
The main goal of this project is to simulate a simplified academic management system and practice:
- structured programming
- working with data collections
- user interaction through console menus

---

## ⚙️ Technologies Used
- C++
- Standard libraries
